一个分裂Alice键盘的制作流程（不保证完成）


可以，按现在定稿，总结成这版。

## 最终产品要求

做一把**极致减薄、无缝、一擦即净、60% 真分体 Alice 机械键盘**。

主力形态是**真分体**，左右完全拉开，小臂自然展开。
便携形态是**合体**，左右半岛咬合，中缝沿键帽错落形成“蛇形阶梯闪电缝”。

键位以你现有键盘为母版，不再拿开源 Alice 当布局母版。现在这套键位已经实践过，原则是：

```text
手掌基本不移动
只移动手指
空白区保留给掌托
不为了填满而加键
```

键位修改要求：

```text
红色大拇指键 → Backspace
左边空格右侧键 → Enter
原 Backspace 位置 → 1U Del + 旋钮
删除方向键
右下空白区继续留空，作为掌托
物理 Win 位置 → Fn
Win 功能实际映射为右 Ctrl
```

主键区：

```text
60% Alice
No Splay
小拇指列大幅下沉 10–15mm
保留当前实际手感为第一优先级
```

## 主控与固件路线

不用 RP2040-Zero。
改用：

```text
nRF52840 Pro Micro / SuperMini 兼容主控
左右各一颗
买 4 留 2，测试后筛板
```

固件路线：

```text
ZMK
BLE 无线分体
一边 central，一边 peripheral
支持蓝牙连接电脑/手机/平板
```

不要走 RP2040 外挂蓝牙。
蓝牙 HID、重连、睡眠、左右分体同步都交给 ZMK。

## 触摸板方案

不再做左右两个侧边触摸板。
改成：

```text
单个中置触摸板
位置：中间区域，红色空格左边
主要由右手操作
```

触摸板候选：

```text
优先：Cirque TM035035，35mm 圆形轨迹板
更好但难买：Cirque TM023023 / 16mm Cirque
备选：Azoteq 小尺寸 trackpad
低价实验：CST816S 类坐标触摸模块
```

触摸板交互：

```text
默认：高灵敏鼠标移动
Fn + 触摸：低灵敏精修 / 滚动
重压：左键
Fn + 重压：右键
```

点击方式用 Force Touch：

```text
触摸板本体不做机械按键槽
触摸板下面放 FSR 压力片
FSR 判断按压
DRV2605L + LRA 马达给点击震感
```

结构要求：

```text
触摸板盖片
↓
触摸板模块
↓
承力背板
↓
FSR 压力传感器
↓
压板
↓
可换垫片
↓
底部硬限位
```

必须预留 0.1 / 0.2 / 0.3 / 0.5mm 垫片空间，方便调预压，不要把间隙一次写死。

## RGB 要求

必须满血 RGB。

RGB 方案：

```text
每个轴位一颗 SK6812 MINI-E / 同类反贴 RGB
嘉立创 SMT 贴装
每半边独立灯链
支持板载满键 RGB
预留外接灯条接口
```

灯不能直射眼睛。
你的轴体是下灯位、有导光柱，所以 PCB 必须按下灯位设计：

```text
灯珠对准轴体导光柱
通过轴体导光柱和键帽扩散
不做裸灯直射
```

因为 PCB/键位是 Alice 弯曲排布，所以不是全板统一方向，而是：

```text
每个轴位以自己的局部坐标为准
轴体旋转多少，RGB 灯珠跟着旋转多少
热插拔轴座也跟着旋转
灯永远贴着轴体导光柱
```

KiCad 最好做一个组合封装：

```text
MX 轴孔
热插拔轴座
下灯位 RGB
定位孔
```

每个键只旋转整个组合封装，避免灯位错。

RGB 电源必须单独设计：

```text
RGB 独立电源轨
每半边 RGB 电源 MOS / 负载开关
每半边大电容焊盘
RGB 数据电阻
必要时加 3.3V → 5V 电平转换
```

预留外接灯条接口：

```text
VLED
GND
DATA_OUT
备用 DATA / GPIO
```

壳体预留：

```text
蛇形中缝导光槽
底边导光槽
触摸板周围导光边
外接灯条线槽
```

## 触觉反馈

每半边保留：

```text
DRV2605L
LRA 线性马达 / iPhone Taptic 拆机马达
```

作用：

```text
触摸板 Force Touch 点击反馈
按键随动震动，可后期打开
层切换 / 蓝牙切换 / 电量提示震动
```

马达供电不要跨左右触点。
每半边本地供电，本地驱动，避免马达瞬态电流拉崩主控。

## 电池与电源

左右半边各一块软包锂电池，带保护板。

电池放在底壳中央腹部空腔。
电池上方不能用碎塑料片，改成连续硬隔片：

```text
FR4
PC
PET
薄绝缘板
```

作用是防止轴体、弹簧、螺丝、结构件戳伤软包电池。

电源结构：

```text
主控/触摸板/FSR/DRV2605L 一路
RGB 单独一路
RGB 可彻底断电
USB 插线时允许满血 RGB
无线时也能开，但续航不保证
```

## 合体与分体结构

分体是主力形态。
合体是便携形态。

合体中缝：

```text
以 6 和 7 内侧边缘作为核心避让基准
最终最小间隙不小于 1.2mm
外壳中缝为蛇形阶梯闪电缝
Alice 几何导致中缝自然上窄下宽
```

合体连接：

```text
磁铁定位
榫舌 / Lip 台阶
金手指 / 弹片触点
```

触点作用：

```text
合体检测
充电 / 维护 / 可选辅助连接
不要承担主力分体通信
```

主力通信走 BLE/ZMK，不靠中缝触点。

触点安全原则：

```text
裸露凸起弹片不要直接暴露高能 5V
电源触点要限流或放在更安全的位置
GND 优先接触
电源后接触
信号最后接触
```

## 脚撑系统

保留 8 个双段脚撑，左右各 4 个。
但它不是 8 个同时展开。

正确定义：

```text
每半边 4 个脚撑 = 姿态组合库
不是四点同时硬支撑
```

每半边：

```text
内侧上/下两个：负责 tenting 帐篷角
外侧上/下两个：负责前后 tilt 倾角
```

实际使用时只展开需要的组合。
外侧底边做大倒角，外沿贴斜面防滑脚垫。

脚撑端部加发泡橡胶垫，用来吃掉桌面误差和制造公差，不是用来强行解决 8 点同时落地。

```text

DRV2605L

JST 电池座
贴片电源开关
MOS / 负载开关
小电阻小电容
```

手焊/后装：

```text
nRF52840 主控板
电池
触摸板
FSR 压力片
LRA 马达
EC11 旋钮
磁铁
弹片
外接灯条
```

PCB 必须预留：

```text

合体检测触点
电池接口
主控排针/半孔
```

## 固件技术路径

ZMK 需要实现：

```text
左右无线分体
现有键位映射
Fn 层
旋钮编码器
满键 RGB
触摸板 pointing device
FSR 压力点击
点击防抖和滞回
DRV2605L 触觉反馈
电池电量
RGB 电源控制
睡眠唤醒
```

FSR 点击逻辑：

```text
press_threshold：按下阈值
release_threshold：释放阈值
释放阈值低于按下阈值，避免抖动
按下瞬间冻结鼠标位移 30–80ms
触发 LRA 短震
```

## 打样顺序

最稳路线：

```text
1. 先用现有键盘照片/实物提取键位中心
2. 确认键位修改和触摸板位置
3. 画左右分体外壳骨架
4. 画 PCB：矩阵、轴座、RGB、主控、触摸板、FSR、DRV2605L
5. 先做一侧 PCB + 一侧触摸板验证
6. 跑通 ZMK、矩阵、RGB、触摸板、FSR
7. 再做完整左右分体
8. 最后优化合体缝、脚撑、电池、灯条、触觉
```

一句话定案：

**这把键盘是“现有实战 Alice 键位 + nRF52840/ZMK 无线真分体 + 满键下灯位 RGB + 中置 Force Touch 触摸板 + LRA 触觉反馈 + 8 脚姿态系统 + 蛇形合体缝”的极薄自定义键盘。**
